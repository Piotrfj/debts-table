# KRD Debts App

A responsive application for searching and sorting debtors, optimized for both mobile and desktop views.

---

## 📦 Install dependencies
```bash
npm install
```


## ▶️ Run in Localhost
```bash
npm start
```

##  🧪 Run tests
```bash
npm test
```

##  ▶️ Build Project
```bash
npm run build
```

##  ▶️ Preview the build
```bash
npm run preview
```
